=============================== Как скомпилировать и подключить учебный пример===============================
1. Установть дистрибутив КОМПАС. При выборе компонентов для установки включить пункт Средства разработки (SDK)
2. Объявить системную переменную KOMPAS_SDK, которая указывает на путь к SDK КОМПАС (например C:\Program Files\ASCON\KOMPAS-3D v20\SDK)
2. Открыть Test.sln в Visual Studio
3. Собрать проект разрядности, сответствующей разрядности КОМПАС 3D (x64 или x86)
4. Подключить собранное приложение Test.rtw в КОМПАС (Приложения->Добавть приложение...)
5. В меню Приложения должно появится подменю Учебный пример
6. Создать новую сборку или деталь в КОМПАС
7. Выполнить команду приложения (Приложения->Учебный пример->Выполнить)
